# Week 1 Public Artifact Launch: What Is AI Visibility?

This launch package turns the first founder/devrel article into a public
category-design artifact.

## Primary Artifact

Post: `posts/what-is-ai-visibility.md`

Primary term: AI Visibility

Canonical definition: AI Visibility is the measurable presence, accuracy,
citation quality, and competitive position of an entity across AI-generated
answers.

## Core Message

OpenVisi is not trying to rename existing search tooling. It defines a
measurement language for the LLM search era: presence, accuracy, citation, and
competition.
