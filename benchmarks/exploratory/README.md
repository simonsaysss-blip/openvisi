# Exploratory Benchmarks

This folder is for small, transparent benchmark snapshots used to harden OpenVisi methodology.

Exploratory benchmarks are:

- non-statistical
- small sample
- methodology-focused
- reproducibility-oriented
- unsuitable for vendor ranking claims

Each benchmark area should document:

- purpose
- inclusion criteria
- sample limitations
- scan command
- methodology version
- date collected
- interpretation cautions

No benchmark data is included yet.
