# QMS Vendors Exploratory Snapshot

This folder is reserved for future exploratory snapshots of public quality-management-system vendor websites.

Purpose:

- inspect machine-readable entity and documentation structure
- compare visibility diagnostics across public website patterns
- identify methodology gaps in regulated or documentation-heavy categories

Limitations:

- no data has been collected yet
- future results should not be treated as vendor rankings
- small samples cannot support statistical conclusions
- OpenVisi does not claim to predict citations or recommendations in LLM products

Required notes for future data:

- methodology version
- scan date
- scan command
- inclusion criteria
- interpretation cautions
