# The OpenVisi AI Visibility Benchmark

Measuring Answer Presence, Entity Clarity, Citation Coverage, and Competitor
Displacement across LLM-powered search surfaces.

OpenVisi benchmark folders are scaffolds for future exploratory methodology work.

They are not statistical studies, ranking datasets, or claims about proprietary LLM behavior. They are intended to make comparison structure explicit before any data is collected.

Rules:

- keep benchmark inputs public and reproducible
- record methodology version
- record crawl date and command
- avoid private or sensitive data
- avoid claims of ranking prediction or citation guarantees
- separate raw runtime reports from curated examples
