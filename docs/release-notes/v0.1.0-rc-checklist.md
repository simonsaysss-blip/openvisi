# v0.1.0 Release Candidate Checklist

Use this checklist before any manual v0.1.0 release action.

## Required Validation

- [ ] `npm run release:check` passed.
- [ ] `npm run demo:mock` passed.
- [ ] `npm run release:rehearse` passed.
- [ ] Tarball contents reviewed.
- [ ] CLI install smoke test passed.

## Claims Review

- [ ] No final metrics claim.
- [ ] No final AI Visibility Score claim.
- [ ] No real provider claim.
- [ ] No production scoring claim.
- [ ] Mock evaluator evidence is described as mock evidence only.

## Release State

- [ ] No `npm publish` has been run yet.
- [ ] No git tag has been created yet.
- [ ] No GitHub release has been created yet.
- [ ] `metrics.json` is not generated by the mock pipeline.
- [ ] `scan-result.json` is not generated by the mock pipeline.
- [ ] Final `report.md` and `report.html` are not generated by the mock pipeline.

## Manual Review

- [ ] `CHANGELOG.md` reviewed.
- [ ] `docs/release-notes/v0.1.0.md` reviewed.
- [ ] `docs/release-rehearsal.md` reviewed.
- [ ] `README.md` release candidate wording reviewed.
- [ ] Package metadata reviewed.
