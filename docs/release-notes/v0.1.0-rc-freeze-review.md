# v0.1.0 RC Freeze Review

This document is the final release candidate freeze review for OpenVisi v0.1.0.

OpenVisi v0.1.0 is a mock-only artifact pipeline release candidate. It is not a production scoring release.

## Release Candidate Status

- Public repo: ready
- v0.1.0 RC: ready after manual review
- Stable v0.1.0: hold until RC review
- Real providers: not included
- Final scoring: not included

## Validated Scope

- AI Visibility core contracts exist.
- Static crawl artifact generation exists.
- Mock evaluator artifact generation exists.
- Measurement input composition exists.
- Metrics draft, metrics review, and metrics finalization guard exist.
- Debug report generation exists.
- Local mock demo verification exists.
- Local release rehearsal exists.
- CLI tarball install smoke has a local rehearsal path.

## Required Commands

These commands must pass before freeze approval:

```bash
npm run release:check
npm run demo:mock
npm run release:rehearse
npm run release:rc-check
```

## Manual Review Checklist

- [ ] README reviewed.
- [ ] CHANGELOG reviewed.
- [ ] Release notes reviewed.
- [ ] Known limitations reviewed.
- [ ] Publish plan reviewed.
- [ ] npm tarball contents reviewed.
- [ ] GitHub Actions reviewed.
- [ ] Issue templates reviewed.
- [ ] Security docs reviewed.
- [ ] `debug-report.md` wording reviewed as non-final report wording.

## Release Blockers

Block the RC if any of these are true:

- `npm run release:check` fails.
- `npm run demo:mock` fails.
- `npm run release:rehearse` fails.
- `npm run release:rc-check` fails.
- Runtime artifact directories remain in the repo root.
- Any documentation claims production AI Visibility scoring.
- Any documentation claims real provider evidence.
- Any documentation claims a final AI Visibility Score.
- Any pipeline generates `metrics.json`.
- Any pipeline generates `scan-result.json`.
- Any pipeline generates final `report.md` or `report.html`.
