# v0.1.0 Known Limitations

OpenVisi v0.1.0 is a mock-only artifact pipeline release candidate.

## Evaluation Limitations

- Mock evaluator only.
- No real LLM provider adapters.
- No OpenAI, Anthropic, Google, Gemini, or other provider API calls.
- No API key support is required for the current pipeline.
- Mock evaluator evidence is deterministic local test evidence, not real LLM evidence.

## Metrics Limitations

- No final `metrics.json`.
- No final AI Visibility Score.
- No `scan-result.json`.
- No production scoring.
- Metrics draft values are transparent intermediate values only.
- Metrics review and finalization guard intentionally block final scoring under mock evidence.

## Report Limitations

- No final `report.md`.
- No final `report.html`.
- `debug-report.md` is an artifact debug report, not a final AI Visibility report.

## Crawl Limitations

- Static crawler only.
- No headless browser rendering.
- No Playwright.
- No Puppeteer.
- JavaScript-heavy pages may not be fully represented.

## Product Limitations

- No SQLite or storage layer.
- No dashboard.
- No hosted SaaS workflow.

## Intended Use

The v0.1.0 artifact pipeline is intended for local OSS demo, artifact contract validation, package smoke testing, and contributor onboarding.
