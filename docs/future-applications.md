# Future Application Areas

OpenVisi starts as an open-source CLI and report toolkit. Future application areas should preserve the OSS core: transparent crawl evidence, explainable scoring, repeatable fixtures, and restrained methodology.

Potential future application areas:

- scheduled diagnostic snapshots
- comparative snapshots across repeated scans
- report template improvements
- documentation workflow integrations
- curated benchmark datasets
- fixture-based analyzer validation
- education and documentation visibility examples
- optional provider-backed interpretation checks with explicit limitations

These ideas are not commitments to hosted workflows, ranking prediction, or citation guarantees.
