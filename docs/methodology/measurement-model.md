# The OpenVisi AI Visibility Model

OpenVisi defines an open-source measurement layer for AI Visibility.

AI Visibility is the measurable presence, accuracy, citation quality, and
competitive position of an entity across AI-generated answers.

The OpenVisi model has four layers.

## 1. Presence Layer

Question: Does AI mention the entity?

Metrics:

- AI Answer Presence
- Mention Rate
- Prompt Coverage
- Answer Share

## 2. Accuracy Layer

Question: Does AI describe the entity correctly?

Metrics:

- Narrative Accuracy
- Entity Clarity
- Positioning Accuracy
- Product Understanding

## 3. Citation Layer

Question: Which sources does AI use to understand the entity?

Metrics:

- Citation Coverage
- Official Source Citation Rate
- Third-party Source Reliance
- AI Citation Signals

## 4. Competitive Layer

Question: Does AI route the user toward competitors?

Metrics:

- Competitor Displacement
- Category Share
- Alternative Recommendation Rate
- Comparison Visibility

## Interpretation

The four layers should be read together. Presence without accuracy can create
misleading visibility. Accuracy without citations may be fragile. Strong
official sources can still lose commercial attention when competitors dominate
comparison answers.
