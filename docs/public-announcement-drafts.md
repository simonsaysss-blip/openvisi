# Public Announcement Drafts

These drafts are intentionally restrained. They position OpenVisi as open-source diagnostic infrastructure, not as a ranking tool or citation guarantee system.

## GitHub Repository Intro

OpenVisi is an open-source toolkit for machine-readable AI visibility diagnostics.

It scans public website signals such as entity clarity, structured data, content chunkability, citation readiness, and technical discoverability, then generates explainable Markdown, JSON, and HTML reports.

The current v0.1.0 release is methodology-first: heuristic scoring, analyzer maturity labels, explainability fields, fixture-based validation, and exploratory benchmark scaffolding.

OpenVisi does not predict rankings or citations in proprietary LLM products. It focuses on transparent, reproducible diagnostics for public website structure.

## Technical Positioning Statement

OpenVisi is open-source AI-readable visibility diagnostic infrastructure.

It evaluates public crawl evidence and reports directional signals across entity clarity, technical discoverability, structured data, content chunkability, citation readiness, and prompt-simulation scaffolding.

The project is designed around explainability: each analyzer exposes score, maturity, detected signals, missing signals, interpretation, suggested structural improvements, and raw evidence.

OpenVisi is not an AI ranking predictor, citation guarantee system, hosted dashboard, or provider-backed LLM evaluation product. It is a local, inspectable foundation for methodology work around machine-readable website visibility.
