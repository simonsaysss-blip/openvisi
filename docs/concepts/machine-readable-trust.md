# Machine-readable Trust

One-line definition: Machine-readable Trust is the set of signals that help
machines assess credibility, authority, and freshness.

## Why It Matters

Trust signals need to be visible and parseable. A claim is harder for AI systems
to use when authorship, source freshness, official status, and supporting
references are unclear.

## How OpenVisi Measures It

OpenVisi checks for structured trust signals, official source consistency,
source freshness, author or organization clarity, documentation depth, and
third-party citation patterns.

Examples:

- structured data
- source freshness
- author or organization clarity
- documentation depth
- third-party citations
- official source consistency

## How Teams Can Improve It

Publish clear ownership and authorship, update important pages, add structured
data, maintain official docs, and make third-party references easy to verify.
