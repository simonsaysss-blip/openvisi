# Entity Clarity

One-line definition: Entity Clarity measures whether AI systems understand what
the company, product, or entity is, what category it belongs to, who it serves,
and how it differs from alternatives.

## Why It Matters

If an entity is unclear, AI answers can describe it vaguely, place it in the
wrong category, or confuse it with competitors.

Common failure modes:

- vague description
- wrong categorization
- outdated positioning
- competitor confusion

## How OpenVisi Measures It

OpenVisi checks whether public pages contain explicit identity, category,
audience, product, location, and differentiation signals.

## How Teams Can Improve It

Write a direct entity definition, use consistent category language, maintain
current product pages, publish comparison context, and align metadata with the
same positioning used in docs and reports.
