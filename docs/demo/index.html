Open-source AI visibility diagnostics

Machine-readable visibility analysis for the LLM search era.

OpenVisi is a CLI-first toolkit for inspecting public website signals that help AI search and LLM-powered discovery systems understand entity clarity, crawlability, structured data, content chunkability, and citation readiness. 

npm install
npm run build
node ./apps/cli/dist/index.js scan https://mavisenglish.com

Run Locally

The public demo is a static snapshot. To generate a fresh report, clone the repository and run the CLI locally. 

git clone https://github.com/simonsaysss-blip/openvisi.git
cd openvisi
npm install
npm run build
node ./apps/cli/dist/index.js scan https://example.com

Architecture Summary

CLI Runs scans and writes reports.
Crawler Collects public pages and site assets.
Core Scores analyzer categories.
Report Generates Markdown, JSON, and HTML.
Fixtures Support repeatable methodology tests.
