Northstar Docs

What the project does

Northstar Docs is an open-source documentation infrastructure toolkit for developer teams. It helps maintainers publish API references, integration guides, and machine-readable project pages.

Who it is for

The toolkit is for developers, maintainers, and documentation teams that need clear public project information.
